## Examples of implementing user authentication with nextjs(SSR), prisma and cookies.

Install dependencies and run dev server
```
yarn install
```
```
yarn run dev
```

Run prisma migrations
```
npx prisma migrate dev
```

Seed test data
```
npx prisma db seed
```
